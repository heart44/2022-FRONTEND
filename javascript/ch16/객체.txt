
객체 (object) : 프로그램에서 인식할 수 있는 모든 대상 (현실 세계 반영하기 위해 나옴)
js에서 객체 : 데이터를 저장하고 처리하는 기본 단위, 웹과 관련된 대상을 모두 객체로 인식
    멤버 필드 - 속성, 프로퍼티(Property)
    멤버 메소드 - 메소드, 객체에 포함돼 있는 함수

js에서 사용하는 객체
    - 문서 객체 모델 (DOM) : 웹 문서 자체도 갳게이고 그 안에 삽입돼 있는 이미지와 링크, 텍스트 필드 등 모두 객체
    - 브라우저 관련 객체 : 웹 브라우저에서 사용하는 정보도 객체로 나타낼 수 있음 (navigator, history, location, screen 등)
    - 내장 객체 : 웹 프로그래밍을 할 때 자주 사용하는 요소가 js안에 미리 객체로 정의돼 있음 (date 등)

클래스, 생성자 함수, <= 틀만 만듬(ex. 붕어빵 틀) =>(만드는 걸 인스턴스[객체화]) 만들어져 나온 붕어빵(객체)
{} (바로 객체화)

프로퍼티 : 객체의 특징이나 속성 (명사담당) - ex. 제조사, 모델명, 색상, 배기량
메소드 : 객체에서 할 수 있는 동작 (동사담당) - ex. 시동걸기, 움직이기, 멈추기, 주차하기

멤버필드는 숨김 (은닉화, 캡슐화) : js에서는 딱히 안함(백엔드에선 함)
    - 외부에서 멤버필드는 직접 쓰기 or 읽기 안 되도록 하는 것
    - 유일하게 오픈돼 있는 멤버필드 : 상수
